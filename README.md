# NPA Tecnologia

> Software world-class para negócios que levam qualidade a sério.

Construímos **landing pages, aplicações web e automações** com stack moderno. Atuamos com clientes de advocacia, gastronomia, beauty, automotivo e saúde — e construímos nossas próprias ferramentas internas.

## Stack padrão

- **Frontend:** Next.js 16, React 19, TypeScript, Tailwind CSS v4
- **Animação:** GSAP, Framer Motion, Lenis
- **Deploy:** Vercel (Edge + Fluid Compute)
- **IA/Automação:** OpenAI, Claude, Gemini, WhatsApp Business API

## Como navegar

- **Projetos de clientes** — filtre por [`client-work`](https://github.com/NPATecnologia?tab=repositories&q=topic:client-work)
- **Ferramentas internas** — filtre por [`npa-internal`](https://github.com/NPATecnologia?tab=repositories&q=topic:npa-internal)
- **Template oficial** — [`repo-template`](https://github.com/NPATecnologia/repo-template) — novo projeto começa daqui
