## `>` Sobre

Construímos software **world-class** para negócios que levam qualidade a sério.
Landing pages sofisticadas, aplicações web robustas e automações com IA — do primeiro commit ao deploy em produção.

Atuamos com clientes de **advocacia, gastronomia, beauty, automotivo e saúde**, e mantemos nossas próprias ferramentas internas. Nosso padrão de entrega tem como referência Apple, Linear, Stripe e Vercel.
